# シート市場 出品機能付き更新版 v4

Supabase DatabaseとStorageを使い、実際にExcel商品を出品できる版です。

## GitHubへの上書き

1. ZIPを解凍
2. GitHubの `sheet-ichiba` を開く
3. `Add file` → `Upload files`
4. 解凍した5ファイルを全部ドラッグ
5. `Commit changes`

## 追加された機能

- ログインユーザーだけ出品可能
- 商品名・カテゴリ・価格・説明を登録
- 商品画像を公開バケットへ保存
- Excel本体を非公開バケットへ保存
- `.xlsx`のみ受付
- 商品画像5MB、Excel10MBまで
- 出品完了後に商品一覧へ即反映
- Supabaseに登録された公開商品を自動表示
- 検索・カテゴリ絞り込みが実データに対応

## 現時点で未実装

- 商品詳細ページ
- 購入決済
- 購入者限定ダウンロード
- 出品商品の編集・削除
- 売上管理

Excel本体は非公開で保存され、現時点では出品者本人以外は取得できません。
